## How To Use

#### Installation

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/adejonghm/dotfiles/main/fedora.sh)"
```

#### NeoVim configuration

Neovim is mandatory

##### Dependencies

- Install npm
- Install ripgrep
- Clone fzf repository
- Use any Nerd Font (Fira Code NF Regular)

_Optional: install starship_
